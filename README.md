---

## ✨ Features

- **Auto-sliding hero** — 6 slides, touch swipe supported
- **Scroll reveal animations** — elements fade in as you scroll
- **3D tilt cards** — mouse-tracked perspective effect
- **Animated skill bars** — fill on scroll
- **Sticky glassmorphism navbar** — blur effect on scroll
- **Contact form** — AJAX submit, no page reload
- **Scroll progress bar** — thin gradient line at the top
- **Fully responsive** — mobile, tablet, desktop
